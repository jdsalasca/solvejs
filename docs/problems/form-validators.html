How to Validate Forms in TypeScript

Use @jdsalasc/solvejs-validators for structured validation results and common rules.

import { validateCellphoneNumber, validateName, validateUrl } from "@jdsalasc/solvejs-validators";

validateCellphoneNumber("+573001112233", { country: "CO" });
validateName("Ada Lovelace");
validateUrl("https://solvejs.dev");

How to Deduplicate Arrays and Compute Percentages Safely

Use @jdsalasc/solvejs-list and @jdsalasc/solvejs-numbers for practical transformations and safe math.

import { unique, sortBy } from "@jdsalasc/solvejs-list";
import { percentChange } from "@jdsalasc/solvejs-numbers";

unique([1, 1, 2, 3]); // [1, 2, 3]
sortBy([{ n: 2 }, { n: 1 }], (x) => x.n); // [{ n: 1 }, { n: 2 }]
percentChange(120, 100); // 20

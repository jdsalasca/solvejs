Format and Parse Dates Safely

Use @jdsalasc/solvejs-date when you need strict parsing, predictable formatting, and UTC-safe date math.

Cookbook Navigation

Recipe

import { parseDateStrict, formatDate, addDays, toIsoDate } from "@jdsalasc/solvejs-date";

const parsed = parseDateStrict("2026-02-07", "YYYY-MM-DD");
if (parsed) {
  const inThreeDays = addDays(parsed, 3);
  formatDate(inThreeDays, "DD/MM/YYYY"); // "10/02/2026"
  toIsoDate(inThreeDays); // "2026-02-10"
}

Why this works: parsing is explicit, formatting tokens are stable, and all date math uses a consistent flow.

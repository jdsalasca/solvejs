SolveJS

Lightweight utilities for solving common JavaScript and TypeScript problems quickly.

Install

npm i @jdsalasc/solvejs

Core Packages

  • @jdsalasc/solvejs-date
  • @jdsalasc/solvejs-string
  • @jdsalasc/solvejs-list
  • @jdsalasc/solvejs-regex
  • @jdsalasc/solvejs-constants

Problem-First Topics

  • How to format dates in JavaScript
  • How to validate cellphone, name, and URL in TypeScript
  • How to deduplicate arrays and compute percentages safely
  • How to convert strings to camelCase or kebab-case

# @jdsalasc/solvejs-date

Zero-dependency date tools for production JavaScript and TypeScript apps.

## Tools Included

- `formatDate`
- `parseDateStrict`
- `parseIsoDate`
- `addDays`
- `startOfDay` / `endOfDay`
- `diffInDays`
- `isLeapYear`
- `daysInMonth`

## Install

```bash
npm i @jdsalasc/solvejs-date
```

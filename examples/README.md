# SolveJS Examples

Runnable starter examples for common framework integrations.

- `nextjs-route-handler/`: request parsing and validation in a Next.js route handler.
- `express-middleware/`: reusable Express middleware for payload validation.
- `env-bootstrap/`: typed environment parsing and startup validation.

Run from repo root:

```bash
npm run build
npm run example:next
npm run example:next:test
npm run example:express
npm run example:express:test
npm run example:env
npm run example:env:test
```
